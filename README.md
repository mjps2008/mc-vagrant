# mc-vagrant
Provision Minecraft server using Vagrant

## Dependencies

* [Vagrant](https://www.vagrantup.com)
* [DigitalOcean Vagrant Provider](https://github.com/smdahlen/vagrant-digitalocean)

## How to use

* Custom your configuration in `config.yml`
* Set your DigitalOcean token in environment variable `DIGITAL_OCEAN_TOKEN`.
* Run `vagrant up`
